Review

rating integer
comment string

Relationships

• User (1) → Order (Many): A user can have multiple orders
• User (1) → Review (Many): A user can write multiple reviews
• Product (1) → Review (Many): A product can have multiple reviews
• Order (1) → OrderItem (Many): An order can have multiple items
• Order (1) → ShippingAddress (1): An order has one shipping address
• Product (1) → OrderItem (Many): A product can be in multiple order items
